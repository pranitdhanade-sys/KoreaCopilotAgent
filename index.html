👤 사용자
파이썬으로 CSV 파일에서
중복된 이메일만 추출하는 함수 만들어줘.

🤖 Copilot
import pandas as pd

def find_duplicate_emails(path: str) -> list[str]:
    df = pd.read_csv(path)
    dup = df[df.duplicated("email", keep=False)]
    return sorted(dup["email"].unique().tolist())

# 사용 예시
print(find_duplicate_emails("users.csv"))

한국어 개발자를 위한 세 가지 핵심

한국어 자연스러움, 한국 개발 생태계 친화, 실무에 바로 쓰는 응답.

🗣️

자연스러운 한국어

어색한 번역체 대신, 실제 한국 개발자들이 쓰는 표현으로 응답합니다. 존댓말/반말 톤 조절도 가능합니다.

FAQ

자주 묻는 질문

GitHub Copilot 유료 구독이 꼭 필요한가요?

Korea Copilot Agent는 GitHub Copilot 위에서 동작하는 지침/에이전트 프로젝트이므로, Copilot 구독(개인/비즈니스/엔터프라이즈)이 필요합니다. 학생/오픈소스 메인테이너는 무료로 사용할 수 있습니다.

다른 한국어 LLM 모델도 지원하나요?

현재는 GitHub Copilot에서 제공하는 모델을 기본으로 사용합니다. 모델 선택은 VS Code의 Copilot Chat에서 변경할 수 있습니다.

회사 코드에 적용해도 안전한가요?

GitHub Copilot Business/Enterprise 플랜에서는 코드가 학습 데이터로 사용되지 않습니다. 회사 정책에 따라 플랜을 선택하세요.

기여하려면 어떻게 해야 하나요?

GitHub 저장소에서 이슈를 열거나, 포크 후 PR을 보내주세요. 한국어 예제, 문서 번역, 버그 리포트 모두 환영합니다.
